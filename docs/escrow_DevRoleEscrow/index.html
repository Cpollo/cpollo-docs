DevRoleEscrow

contract DevRoleEscrow

is BaseEscrow

This is the escrow that holds funds for development. Only registered Developers are allowed to receive token funds. When there are signals of a scam, Cpollo will freeze the funds and start the auditing process. If the signals are determined to be correct, Cpollo will return funds to the team wallet.
