TokenDynamicPriceCrowdsale

contract TokenDynamicPriceCrowdsale

is TokenCrowdsale, Secondary

Extension of Crowdsale contract that allows crowdsales to dynamically set the price in tokens. This can be used to create OTC crowdsales.

Source: token-crowdsale/price/TokenDynamicPriceCrowdsale.sol
